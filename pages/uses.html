Usage

Main Setup

I use EndeavourOS on a daily basis, which is an Arch-based, terminal-centric distribution. I've opted for the i3 window manager because I find its tiling window functionality helps me stay organized.

For specific security-related tasks, I turn to Kali Linux. When I require a minimalistic distribution, I prefer #!++.

Software Choices

I strive to use FOSS (Free and Open-Source Software) as much as possible. Due to my work, I am sometimes compelled to use non-free software, but I make an effort to minimize this. I also prefer self-hosted solutions over cloud-based ones.

  • My default web browser is Firefox, and I take the necessary security measures to avoid Mozilla's tracking. You can use this generator to get started and then follow this guide to enhance your privacy.
  • I use VSCodium as my integrated development environment (IDE). VSCodium is Visual Studio Code without Microsoft's telemetry and bloat.
  • For text editing, I prefer NeoVIM.

Technology Stack

I primarily work as a web developer, specializing in PHP and JavaScript. However, I occasionally explore other programming languages for specific use cases.
